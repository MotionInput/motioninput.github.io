Updates the colour of an extremity circle when the extremity is activated or deactivated by updating the list of extremity circles. If an extremity is activated, the circle turns green. If an extremity is deactivated, the circle turns red. Also adds the number of times the extremity has been triggered.

  • repeats (str) – Number of times the extremity has been triggered
